## About The Project

Projet réalisé dans le cadre de l'unité d'enseignement **Modélisation Géométrique** en Master 2 Informatique - ID3D à l'Université Claude Bernard Lyon 1.

### Génération de surfaces maillées
L’objectif est de générer des surfaces maillées à partir d’un ensemble de points ou de courbes
de contrôle.

### Surfaces de révolution et d’extrusion
L’objectif est de générer des surfaces d’extrusion à partir d’une courbe génératrice.

<p align="right">(<a href="#readme-top">back to top</a>)</p>

### Built With

The project is in C++. The libraries used for project are listed below : 

* [GKit](https://perso.univ-lyon1.fr/jean-claude.iehl/Public/educ/M1IMAGE/html/index.html) : Used for visualisation. It is used as an external library, check the [prerequisites](#prerequisites) section. 
* [ImGUI](https://github.com/ocornut/imgui) : Used to managed the UI of the application. It is located in the *vendor* directory. 
* [exprtk - C++ Mathematical Expression Toolkit Library](https://github.com/ArashPartow/exprtk) : Used to parse string that represent mathematical expression. It is located in the *vendor* directory.
